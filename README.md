## Overview


### The challenge

The project was prepared for three breakpoints, using the mobile first convention. Project based on design from [Colorlib.](https://colorlib.com/)

Breakpoints:

- mobile ( to 768px )
- tablet (from 768px to 1200px)
- desktop (from 1200px)
